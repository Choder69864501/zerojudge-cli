# zerojudge-cli

___

## Introdution

這是一個能夠用python去與zerojudge互動的簡易小程式 

____ 

## Install

下載，然後安裝python的要用的pacakge:requests,beautifulSoup4

___

## Feature
1. 可以看即時動態
2. 可以提交程式碼
3. AC，TLE等等有高亮
